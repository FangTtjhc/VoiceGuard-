# VoiceGuard-
VoiceGuard 是一个基于 Flask 框架开发的现代化、智能化的访客登记系统。它创造性地将语音识别 (STT)、大语言模型 (LLM) 结构化数据提取、声纹生物识别和自动化通知相结合，旨在提供一个安全、高效、无纸化的访客信息采集与管理解决方案。  本项目已正式开源，旨在推动 AI 与生物识别技术在实际安全管理场景中的应用与交流。
核心功能与技术亮点1. 🎤 语音驱动的访客登记 (STT & LLM)多格式音频支持: 支持多种音频格式上传。语音转文字 (STT): 采用 Google Speech Recognition API 对访客语音进行精确转录。LLM 结构化提取: 利用大型语言模型（如 GPT-4o-mini）的强大能力，将自由文本的访客信息（姓名、单位、事由等）自动解析并结构化为标准 JSON 格式数据。2. 🛡️ 声纹生物识别与身份核验MFCC 声纹提取: 使用 librosa 库提取语音的 梅尔倒谱系数 (MFCC) 作为声纹特征向量。动态声纹比对: 基于 欧氏距离 进行声纹比对。新访客：自动注册并建立声纹 ID。重复访客：如果声纹匹配成功，系统自动补全历史记录中的准确信息，无需人工修正。声纹数据库管理: 将访客的声纹 ID 和特征向量存储在本地 voiceprint_db.json 中，确保数据的自主可控。3. 🖥️ 流程自动化与数据闭环人性化修正界面: 对于 LLM 提取不准确或声纹匹配失败的数据，提供一个基于 HTML 模板的修正页面，允许管理员在保存前快速校对。自动化 CSV 记录: 所有登记成功的记录（包括声纹匹配成功的自动记录）都将实时保存到本地 CSV 文件中，方便后续数据分析和审计。邮件通知集成: 当访客记录成功保存后，系统将自动触发邮件通知（需配置 email_test.py），确保相关负责人能及时获得访客信息。架构依赖本项目主要依赖以下关键库和技术：类别库/技术用途Web 框架Flask提供轻量级的 Web 服务和路由。LLM 交互requests用于调用第三方 LLM API 进行数据结构化。语音处理librosa, pydub, speech_recognition声纹提取、音频标准化、语音转文本。数据处理pandas, numpy数据结构化存储、CSV 读写。系统FFmpeg/FFprobe音频格式转换和处理工具（需单独安装和配置路径）。快速部署环境配置: 安装 Python 依赖，并确保安装了 FFmpeg 并配置了路径（见 app.py 中 FFMPEG_BIN_DIR 配置）。API 配置:配置 LLM API 密钥：设置环境变量 OPENAI_API_KEY。配置邮件服务：确保 email_test.py 中的 SMTP 账号和密码已通过环境变量安全加载。运行:HTTP: python app.pyHTTPS: 确保 server.crt 和 server.key 证书文件存在于根目录。许可证本项目采用 MIT 许可证 开放源代码。欢迎 fork、提交 Pull Request，共同改进！
